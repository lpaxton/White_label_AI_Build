RECOMMENDED APPROACH
================================================================================

For most users, we recommend:

1. Use the Python scraper (fidelity_scraper.py)
   - Most reliable
   - Best results
   - Easy to use

2. Install dependencies:
   pip install requests beautifulsoup4 lxml

3. Run the scraper:
   python fidelity_scraper.py

4. Review results:
   - Open the CSV file in Excel/Google Sheets
   - Or use the JSON files for programming
