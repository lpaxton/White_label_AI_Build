Welcome to Article Generator! 🎯

This tool uses Retrieval-Augmented Generation (RAG) with Ollama to generate articles in your writing style.

🚀 Quick Start:

  1. Bulk Upload Articles: Add your .txt files (50+ supported!)
  2. Test Connection: Make sure Ollama is running
  3. Enter Prompt: Describe what you want to write about
  4. Generate: Let AI create content in your style!

💡 Bulk Upload Tips:
• Select all files: Ctrl+A in file browser
• Drag entire folders with .txt files
• System processes in batches for efficiency
• More articles = better style matching!

🎯 How it works:

• Your articles are chunked and stored in a vector database

• When you provide a topic, it finds similar content from your articles

• Ollama generates new content using those examples as style guides

💡 Tips: Use topics similar to your existing articles for best results!
